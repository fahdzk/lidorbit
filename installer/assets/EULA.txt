2. FUNCTIONAL CAPABILITIES (WHAT THE SOFTWARE DOES)
--------------------------------------------------------------------------------
LidOrbit is a desktop utility designed to manage and override default system 
standby, sleep, and hibernate behaviors. Key operations include:
  - Core Sleep Bypass: When toggled to ON, the Software blocks standard display 
    and system sleep behavior.
  - Platform-Specific Tweaks:
      * Windows: Automatically queries and backs up active standby and hibernate 
        plan timeouts, temporarily overrides them to '0' (never sleep), and 
        modifies the native Windows Lid-Close Action configuration to 'Do Nothing'.
      * macOS: Spawns and keeps active a background 'caffeinate' terminal command.
  - State Restoration: On graceful exit or toggle switch closure, LidOrbit 
    attempts to restore all original power timeouts and configurations.
  - Battery Safety Alarm: Includes a background loop that monitors battery levels. 
    If battery capacity falls to 15% or below and the device is unplugged, the 
    Software emits a dual-beep audio alert and displays system notifications to 
    prompt immediate recharging. This alarm repeat interval is user-configurable, 
    subject to a strict safety minimum limit of 8 minutes.
